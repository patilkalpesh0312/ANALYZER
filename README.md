## 📖 About

**NumPy Analyzer** is an interactive **menu-driven Python project** built using **NumPy**. It enables users to create arrays, perform mathematical operations, combine or split arrays, search, sort, filter data, and compute statistical measures through an easy-to-use command-line interface.

---

## ✨ Features

| 🚀 Feature | 📋 Description |
|:---------:|:---------------|
| 📦 Array Creation | Create **1D, 2D & 3D** NumPy arrays |
| ➕ Math Operations | Addition, Subtraction, Multiplication & Division |
| 🔀 Combine & Split | Merge or Split arrays |
| 🔍 Search & Sort | Search, Sort & Filter arrays |
| 📊 Statistics | Sum, Mean, Median, Standard Deviation & Variance |

---

## 📚 NumPy Concepts

✅ Array Creation • Reshaping • Mathematical Operations • Element-wise Computation • Vertical Stacking (`vstack`) • Array Splitting • Searching (`where`) • Sorting (`sort`) • Filtering • Aggregate Functions • Statistical Functions
